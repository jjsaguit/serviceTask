Microsoft Academic Search

A service for searching academic publications on Microsoft Academic Search.

Parameters

titleStringThe title of publication to search

Results

resultsObjectDataType("java.util.Map")List of top 10 publications found

Handler

org.msho.app.MicrosoftAcademyWorkItemHandler("AppID")

Requesting an AppID
To identify each application using the MAS API, an AppID is required. To request an AppID provide the following information in an email request to acadapi@microsoft.com:

  • Contact Person
  • Contact Email
  • Project URL
  • Confirmation that you agree to the MAS Terms of Use and MS Terms of Use.
    • E.g. "We have read and agreed with the Microsoft/MAS Terms of Use"
    • Include a link to the MAS Terms of Use, MS Terms of Use.

Ref: Microsoft Academic Search API User Manual
